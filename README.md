
# 🎬 Автотесты для Кинопоиска

## 📌 Описание проекта
Этот проект содержит автоматизированные UI и API тесты для веб-приложения Кинопоиск.
Тесты написаны с использованием паттерна Page Object и покрывают ключевые пользовательские сценарии.
